APPNODE免费版站点限额提升至10个

发表于 2020-03-22 更新于 2021-07-13 分类于 福利活动 Waline： 阅读次数：

AppNode是一个国产的Linux服务器管理面板，曾经免费版只支持3个网站，对于个人站长来说，勉勉强强，折腾一下就不够用了，今天在某群看到伊卡洛斯Dalao分享，所以我就跑来水了一篇

AppNode安装比较简单，复制粘贴下面的命令即可

INSTALL_AGENT=1 INIT_SWAPFILE=1 bash -c "$(curl -sS http://dl.appnode.com/install.sh)"

以下是我从官网偷录抄袭的GIF（小声bb）

官网规定的安装条件

操作系统： 
CentOS 6.x 32/64位
CentOS 7.x 64位
（暂不支持 Ubuntu、Debian）。
内存要求： 
建议可用内存在 512MB 以上。

至今还没不支持Ubuntu就一点不香了

我对Appdode的一些个人主观看法

功能挺多，基本能满足，界面设计不够方便，看起来比较复杂，功能上比隔壁btpanel好，界面上差于btpanel，对于集群管理什么的，应该比较方便吧，不过btpanel的集群管理我是没用过哈哈，本来就一台服务器，也用不着

更多Appnode相关的可以看Qi Dalao写的：https://wzfou.com/appnode/

Appnode官网：https://www.appnode.com/

更多服务器面板等看这：https://blog.shiyunhong.com/2750.html

近期对于新型冠状病毒肺炎的防治

国家给出态度了，治疗费用全部免除，大力布防疫情扩散。

医学界给出态度了，80多岁的院士不在家和亲人过春节毅然奔赴重灾现场，

很多医院取消春节休假。

武汉人民付出了，出行都受到了限制。

而武汉之外的很多人却摆出了一副事不关己高高在上的态度。

抖机灵 

发段子

看笑话

不重视

“怕什么啊，爱情都轮不到我，肺炎能轮到我？”

“天天约炮，艾滋都不怕的能怕肺炎？”

蹦迪蹦傻了的才能说出这种话，说出这种话的人可想而知每天都生活在什么样的圈子里。

病的不是你家人你就觉得这是很好玩的事情了吗？

”丧尸之城？“

意外被感染的武汉人也很无辜,试想如果是你得了病,却被别人比喻成丧尸你是什么心情呢?

“你好我是医生，最近肺炎严重，请大家把胸部照片发给我看一看，因为从皮肤可看出你有没有得肺炎，严重的话可能还要看一下下半身。”

不知道是哪个小学文盲写的段子，你们疯狂转发，还以为自己幽默了。。。

怎么说呢，对疾病的重视不光是对自己负责人，也是对自己的家人，朋友，爱人的负责人,发这种段子？

周围不了解病毒传播的人跟着你一起轻视，你无所谓，老人孩子怎么办？

同时也对医护人员负担的减轻，每每网上有一些让人引起公愤的事情时,感觉人人都有正义感,义愤填膺。

怎么这种利己利人的小事反而不重视呢?私生活正常的不用担心艾滋病

戴口罩的成为了被笑话的对象,被嘲笑怕死。

不做防护措施的把“胆大”当成骄傲的资本。

有时候意外比明天先来临。 希望我朋友圈和空间的每一位都不会成为这个倒霉蛋

但是预防疾病，人人有责。
