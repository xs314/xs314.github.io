通过宝塔进行安装JPress

发表于 2019-06-30 更新于 2021-07-13 分类于 网站源码 Waline： 阅读次数：

JPress简介

JPress是一款优秀的国产程序，模仿Wordpress开发，基于Java、支持分布式部署，可以横向扩展支持任何流量的大型产品。同时，可以通过新增模块、插件的方式做任何类型的网站、APP和微信小程序，包括但不限于官网、营销页、博客、论坛、商城等，基本符合国人胃口。

本文主要内容概括

使用宝塔面板（bug tower） 安装JPress

检测相关环境

  • 确保安装宝塔（bug tower)
  • nginx ： 1.14
  • tomcat ： 8.5
  • mysql ： 5.6

创建网站

在宝塔后台，通 网站 > 添加网站 创建一个新的网站。

创建网站的时候需要注意的是，创建mysql数据库的时候，版本要选择 utf8mb4 编码。

在宝塔后台的 网站 里，点击网站域名，在 tomcat 菜单里，启用 tomcat 功能。

上传官方源码

链接: https://pan.baidu.com/s/107WQ-y9EbYrBMEuXgfYZwQ 提取码: j7x3

访问网站走jpress自动安装过程

访问你的域名，JPress自动引导进行安装，在JPress安装向导的过程中，只需要填写宝塔创建完毕的数据库账号和密码即可，如PHP程序般简单。

简介

English version

YUMI (Your Universal Multiboot Installer), is the successor to our MultibootISOs. It can be used to create a Multiboot Bootable USB Flash Drive containing multiple operating systems, antivirus utilities, disc cloning, diagnostic tools, and more. Contrary to MultiBootISOs which used grub to boot ISO files directly from USB, YUMI uses syslinux to boot extracted distributions stored on the USB device, and reverts to using grub to Boot Multiple ISO files from USB, if necessary.

中文简介（by 机翻）

YUMI（您的通用多重引导安装程序）是我们的MultibootISOs的继承者。 它可用于创建包含多个操作系统，防病毒实用程序，光盘克隆，诊断工具等的多重引导可启动USB闪存驱动器。 与使用grub直接从USB引导ISO文件的MultiBootISO相反，YUMI使用syslinux来引导存储在USB设备上的提取的分发，并在必要时恢复使用grub从USB引导多个ISO文件。

Linux下解压7z压缩包

发表于 2019-06-17 更新于 2021-07-13 分类于 技术教程 Waline： 阅读次数：

前言

在linux中，通常是不支持7z压缩包的解压的，需要另行安装软件进行解压，p7zip就是一款开源的解压缩软件（也是著名7-zip的linux移植版）

开始

Centos

yum install p7zip

Ubuntu/Debian等

apt install p7zip

其他

wget https://downloads.sourceforge.net/project/p7zip/p7zip/16.02/p7zip_16.02_src_all.tar.bz2
tar -jxf p7zip_16.02_src_all.tar.bz2
bash install.sh

基本命令

压缩

7za a 压缩包.7z 被压缩文件或目录

解压

#将压缩包解压到指定目录，注意：指定目录参数-o后面不要有空格
7za x 压缩包.7z -o解压目录
#将压缩包解压到当前目录
7za x 压缩包.7z

Glyptodon Enterprise-基于Apache的“桌面云”安装教程

发表于 2019-04-30 更新于 2021-07-13 分类于 技术教程 Waline： 阅读次数：

 

前言

在几个月前，https://demo.glyptodon.com这个Demo就挺火了，但是迟迟没有搭建教程,经过对小石的摸索，找到了教程

准备

一台Centos6.9+服务器，再加一台Windows服务器

  1. 在官网注册：https://enterprise.glyptodon.com
注册都懂的，注册购买
  1. 转到download,看到自己的用户名和密码，用小本本记下来

正式教程

视频教程

源于官方网站，纯英文

如观看不便，请访问：https://www.bilibili.com/video/av48261276/

文字教程

小石纯手打

sudo vi /etc/yum.repos.d/glyptodon.repo

i

[glyptodon]
name=Glyptodon Enterprise
baseurl=https://你的用户名:你的密码@enterprise.glyptodon.com/release/1/el7/$basearch/
enabled=1
gpgcheck=1
repo_gpgcheck=1
gpgkey=https://enterprise.glyptodon.com/release/RPM-GPG-KEY-glyptodon-release

:wq

创建以上文件

sudo yum install epel-release

sudo yum install @glyptodon-guacamole

sudo yum install tomcat

sudo usermod -aG guacamole tomcat

sudo ln -s /usr/share/guacamole/guacamole.war /var/lib/tomcat/webapps/

sudo systemctl start guacd tomcat
sudo systemctl enable guacd tomcat

sudo service guacd start
sudo service tomcat start
sudo chkconfig guacd on
sudo chkconfig tomcat on

安装相关软件

http://你的服务器IP:8080/guacamole/

访问以上网址

sudo vi /etc/guacamole/user-mapping.xml

编辑以上配置文件

<user-mapping>

    <!-- Example user with a single connection and plaintext password -->
    <!--
    <authorize username="用户名1" password="密码1">
        <protocol>vnc</protocol>
        <param name="hostname">Vnc连接地址</param>
        <param name="port">vnc连接端口</param>
        <param name="password">vnc连接密码</param>
    </authorize>
    -->

    <!-- Example user with multiple connections and an MD5-encoded password
         of "PASSWORD" -->
    <!--
    <authorize username="用户名2" password="密码2的md5值，如无需md5加密请把后面的 (encoding="md5") 干掉"
               encoding="md5">

        <connection name="随便一个名称">
            <protocol>vnc</protocol>
            <param name="hostname">vnc连接地址</param>
            <param name="port">vnc连接端口</param>
            <param name="password">VNC密码</param>
        </connection>

        <connection name="随便一个名称，如xiaoshi is very handsome">
            <protocol>rdp</protocol>
            <param name="hostname">rdp连接地址</param>
            <param name="username">RDP用户名</param>
            <param name="password">RDP密码</param>
        </connection>

    </authorize>
    -->

</user-mapping>

呐，这是小石对重要部分的解释

配置完成后再访问：http://你的服务器IP:8080/guacamole/

登陆，sucess!
